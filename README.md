# Alice — The Laptop Finder

![Zoho SalesIQ](https://img.shields.io/badge/Zoho_SalesIQ-Chatbot-red?style=flat)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

An intelligent chatbot built on Zoho SalesIQ that helps users find the perfect laptop based on their specifications and preferences. Alice guides users through a conversational flow to recommend laptops tailored to their needs.

## Features

- **Conversational Interface** — Natural chat-based laptop recommendation
- **Guided Search** — Alice asks about your requirements step by step
- **Self-Browse Option** — Direct link to laptop buying guide for manual browsing
- **Context-Aware** — Maintains conversation context across interactions

## Tech Stack

| Component | Technology |
|-----------|-----------|
| **Chatbot Platform** | Zoho SalesIQ |
| **Bot Logic** | Zoho SalesIQ Scripting (Deluge-like) |
| **Handlers** | Context Handler, Message Handler, Trigger Handler, Failure Handler |

## Project Structure

```
├── Context Handler.py # Main conversation logic & laptop recommendations
├── Message Handler # Handles incoming user messages
├── Trigger Handler # Triggers for initiating conversations
├── Failure Handler # Error handling
├── dp.jpg # Bot avatar
└── Screenshot*.png # Application screenshots
```
